https_for_google_search_result
==============================

此项目为CHROME应用，尚未发布至应用商店。
在景德镇，谷歌搜索结果页面是可以打开的，但点击结果中的链接跳转时经常就卡在那里了，手动在其前面加个HTTPS就可正常跳转。
此应用将谷歌搜索结果链接的HTTP换为HTTPS，在打开搜索结果链接的同时使用此应用即可，这样就不用为搜索结果链接手写HTTPS啦。

==============================

安装方法：
1.打开CHROME浏览器；
2.工具->扩展程序->勾选“开发者模式”->载入正在开发的扩展程序；
3.选择此项目的Https_for_Google_search_result文件夹导入即可。
